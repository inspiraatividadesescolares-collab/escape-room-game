# 🏚️ O Casarão Abandonado - Escape Room

Um jogo de escape room interativo com 25 desafios de raciocínio lógico para jogar em dispositivos móveis e desktop.

## 📋 Características

- **25 Desafios** divididos em 5 níveis de dificuldade
- **Narrativa envolvente** com 7 telas de contextualização
- **Sistema de cronômetro** com 1 minuto por questão
- **Pistas desbloqueáveis** ao completar cada nível (4+ acertos)
- **Desafio final** - descubra a senha para escapar
- **Interface responsiva** otimizada para telas pequenas em paisagem
- **Suporte offline** com PWA e Service Worker
- **Efeitos sonoros** personalizáveis

## 🚀 Como Instalar

### No Android:
1. Abra o arquivo `index.html` no navegador Chrome
2. Clique no menu (⋮) → "Instalar aplicativo" ou "Adicionar à tela inicial"
3. O jogo será instalado como app PWA

### No iOS:
1. Abra em Safari
2. Clique em Compartilhar → Adicionar à tela inicial
3. Nomeie como "Escape Room"

### No Desktop:
1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Use o botão "Tela Cheia" para melhor experiência
